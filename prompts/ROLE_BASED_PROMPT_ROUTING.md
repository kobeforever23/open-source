# Role-Based Prompt Routing
**AS-OF:** 2026-03-05 13:56:42 EST

## Analyst
- Prioritize: data controls, reconciliations, monitoring prompts.
- Output: accurate first-pass analysis with evidence links.

## Associate / AVP
- Prioritize: scenario analysis, limit monitoring, issue remediation prompts.
- Output: decision-ready packages with challenge notes.

## VP / ED
- Prioritize: executive brief, committee pack, cross-risk synthesis prompts.
- Output: decisions, trade-offs, escalation pathways.

## CRO / Senior governance
- Prioritize: strategic risk posture and enterprise constraints.
- Output: top risks, mitigation portfolio, governance actions.
