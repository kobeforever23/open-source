# Prompt Standards for CIB Risk
**AS-OF:** 2026-03-05 13:56:42 EST

## Standard
Every prompt must enforce: actionability, citations, audit trail, uncertainty labeling, and owner-based execution.

## Required components
1. Objective and scope
2. Input contract
3. Output contract
4. Control checks
5. Red-team checks
6. Scoring rubric

## Release checklist
- Prompt tested on at least 3 representative cases.
- Citation behavior verified for regulatory claims.
- Escalation logic validated against governance policy.
- Output readability checked by analyst and VP-level reviewer.
