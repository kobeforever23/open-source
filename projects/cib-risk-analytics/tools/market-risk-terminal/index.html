Method

sigma_i = delta_i * vol_i * sqrt(h)
sigma_p = sqrt(sigma^T * Corr * sigma)
VaR = z(alpha) * sigma_p; ES = phi(z)/(1-alpha)*sigma_p
MC PnL_i = delta_i*shock_i + 0.5*gamma_i*shock_i^2
Kupiec LR = -2 ln[(1-p)^(n-x) p^x / (1-x/n)^(n-x) (x/n)^x]
Backtest exceptions use threshold PnL < -VaR for each model variant.
