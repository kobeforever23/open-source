# Starter Templates

Basic templates and boilerplate for getting started with common risk analytics patterns.

## Contents

| Template | Description |
|---|---|
| `basic-risk-dashboard.html` | Minimal single-page risk dashboard skeleton |
| `sample-data-loader.py` | Python utility for loading and validating CSV risk data |

## Usage

These templates are intentionally minimal. They provide a starting point for building risk-oriented tools and can be extended as needed.

### Dashboard Template

Open `basic-risk-dashboard.html` in any browser. No dependencies required.

### Data Loader

```bash
python3 sample-data-loader.py --input data.csv
```

## Notes

- These are educational starting points, not production tools
- For production-grade implementations, see the other projects in this repository
