# EQFIN Shock Calibrator
**AS-OF:** 2026-03-05 14:03:09 EST

Standalone market-shock calibration tool.

## Files
- `index.html`: browser GUI for shock solving.
- `data.js`: constituent and weight data.
- `calibrator_gui.py`: optional Python GUI helper.
- `spx_ndx_data.py`: data processing support.

## Run
Open `index.html` directly, or run the suite server from `../` and open `/eqfin-shock-calibrator/index.html`.
