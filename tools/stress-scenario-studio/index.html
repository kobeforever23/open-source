Methodology

PnL_i = delta_i*shock_i + 0.5*gamma_i*shock_i^2 + vega_i*volShock_i
Total PnL = Σ PnL_i
Reverse stress scale m solves: A*m^2 + B*m = targetLoss
where A = 0.5*Σ(gamma_i*shock_i^2) and B = Σ(delta_i*shock_i + vega_i*volShock_i).
