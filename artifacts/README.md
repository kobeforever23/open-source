# Artifacts

Archived and supporting materials that are not part of the core prompt/skill/tool libraries.

## Structure
- `dashboards/`: legacy HTML dashboards and prototypes
- `reports/`: exported PDF/DOCX/MD report outputs
- `reference/`: one-off markdown references and cheat sheets

## Guidance
- Treat these files as reference snapshots unless promoted into `tools/` or `briefs/`.
- Prefer adding new production prompts to `prompts/` and production workflows to `tools/`.
