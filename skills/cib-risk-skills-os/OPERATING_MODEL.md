# CIB Risk Skills OS Operating Model
**AS-OF:** 2026-03-05 13:24:30 EST

## Mission
Run CIB Risk as a repeatable, continuously improving skill system that is practical for analysts and defensible to senior management, audit, and regulators.

## Operating loops
1. Execution loop (daily/weekly): run prompts + skill playbooks, deliver decisions.
2. Learning loop (weekly): drills, retrospectives, skill-gap closure.
3. Update loop (weekly/monthly): source scan, diff review, controlled merges.

## RACI
- Risk domain owners: content accuracy and decision quality.
- Control owners: policy alignment and evidence quality.
- Data/tech owners: reproducibility, lineage, and reliability.
- Library steward: index integrity and change governance.

## Standards
- Citation and confidence: see `VERIFICATION_POLICY.md`.
- Update cadence and PR rules: see `UPDATE_PLAYBOOK.md`.
- Learning execution: see `LEARNING_SYSTEM.md`.
